# MD to PDF - Веб-версия

Рефакторинг оригинального Electron-приложения в веб-версию с сохранением всей логики, дизайна и уникальных возможностей.

## Структура проекта

```
web-app/
├── index.html          # Главный HTML файл
├── package.json        # Зависимости и скрипты
├── vite.config.js      # Конфигурация Vite
├── src/
│   ├── main.js         # Точка входа приложения
│   ├── modules/        # Модули (логика)
│   │   ├── editor.js       # Редактор CodeMirror 6
│   │   ├── preview.js      # Превью Markdown
│   │   ├── toolbar.js      # Панель инструментов
│   │   ├── file-io.js      # Работа с файлами (адаптировано для веба)
│   │   ├── state.js        # Управление состоянием
│   │   ├── ai-chat.js      # AI чат
│   │   ├── ai-feedback.js  # AI обратная связь
│   │   ├── ai.js           # AI логика
│   │   ├── editor-theme.js # Темы редактора
│   │   ├── search-replace.js # Поиск и замена
│   │   └── word-counter.js # Счетчик слов
│   └── styles/         # CSS стили
└── dist/               # Сборка для продакшена
```

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Особенности веб-версии

### Адаптированные функции:
1. **Открытие файлов**: Через стандартный `<input type="file">` вместо Electron dialog
2. **Сохранение файлов**: Используется Blob API и скачивание через браузер
3. **Экспорт PDF**: Через встроенную функцию печати браузера (Ctrl+P)

### Сохраненные функции:
- ✅ Живой превью Markdown с подсветкой синтаксиса
- ✅ Синхронизация скролла между редактором и превью
- ✅ Поддержка формул MathJax (LaTeX)
- ✅ AI-помощник (чат и форматирование)
- ✅ Темы (светлая/темная)
- ✅ Горячие клавиши
- ✅ Статистика документа (слова, символы, время чтения)
- ✅ Масштабирование интерфейса
- ✅ Поиск и замена

## Отличия от Electron-версии

| Функция | Electron | Web |
|---------|----------|-----|
| Открытие файлов | Нативный диалог | Input element |
| Сохранение файлов | ФС напрямую | Blob download |
| Экспорт PDF | Puppeteer/Chrome | Browser print |
| Кастомная шапка окна | Да | Нет (стандартная) |

## Технологии

- **Vite** - сборщик
- **CodeMirror 6** - редактор кода
- **Markdown-it** - парсинг Markdown
- **Highlight.js** - подсветка кода
- **MathJax 3** - рендеринг формул
- **GitHub Markdown CSS** - стили превью
