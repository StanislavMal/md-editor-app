## 1. 📊 WORD/CHARACTER COUNTER - СЧЕТЧИК СЛОВ

**Цель:** Отображение статистики документа в статус-баре

**Технические детали:**
- Расположение: Нижняя панель (статус-бар) рядом с масштабом
- Показывать: слова, символы (с пробелами/без), строки, время чтения
- Обновление: realtime при вводе текста

**Реализация:**
1. **Новый модуль:** `src/renderer/modules/word-counter.js`
2. **Функции:**
   - `countWords(text)` - подсчет слов
   - `countChars(text, includeSpaces)` - подсчет символов
   - `estimateReadingTime(wordCount)` - оценка времени чтения
   - `updateStats()` - обновление UI

3. **UI компоненты:**
   - HTML: `<div class="status-bar">` в `index.html`
   - CSS: `src/renderer/styles/components/_status-bar.css`
   - Обновление: `toolbar.js` - добавить инициализацию

4. **Интеграция:**
   - Подключить к `editor.js` - слушать изменения текста
   - Событие: `updateWordCount` при каждом изменении

**Оценка времени:** 2-3 дня

## 2. 🔍 SEARCH & REPLACE - ПОИСК И ЗАМЕНА

**Цель:** Полнофункциональный поиск с заменой

**Технические детали:**
- Горячие клавиши: Ctrl+F (поиск), Ctrl+H (замена)
- Поддержка: regex, case-sensitive, whole words
- Навигация: стрелки вверх/вниз по результатам

**Реализация:**
1. **Новый модуль:** `src/renderer/modules/search-replace.js`
2. **Функции:**
   - `openSearchDialog()` - открытие диалога
   - `performSearch(text, pattern, options)` - поиск
   - `replaceAll(text, oldPattern, newText)` - замена всех
   - `replaceNext()` - замена следующего

3. **UI компоненты:**
   - Модальный диалог с полями ввода
   - Кнопки: Find Next, Replace, Replace All
   - CSS: `src/renderer/styles/components/_search-dialog.css`

4. **Интеграция:**
   - CodeMirror search API
   - Горячие клавиши в `editor.js`
   - Обработчики в `toolbar.js`

**Оценка времени:** 3-4 дня

## 3. 💾 AUTO-SAVE - АВТОМАТИЧЕСКОЕ СОХРАНЕНИЕ

**Цель:** Сохранение черновиков и восстановление при сбое

**Технические детали:**
- Интервал: каждые 30 секунд при изменениях
- Хранение: localStorage / indexedDB
- Восстановление: при загрузке приложения

**Реализация:**
1. **Новый модуль:** `src/renderer/modules/auto-save.js`
2. **Функции:**
   - `startAutoSave()` - запуск таймера
   - `saveDraft(content, timestamp)` - сохранение
   - `loadDraft()` - загрузка черновика
   - `clearDraft()` - очистка при сохранении

3. **Логика:**
   - Проверка на изменения перед сохранением
   - Debounce для оптимизации
   - Уведомления пользователю

4. **Интеграция:**
   - Подключить к `state.js` - отслеживание изменений
   - UI: уведомление при восстановлении черновика

**Оценка времени:** 2 дня

## 4. 📖 TABLE OF CONTENTS - ОГЛАВЛЕНИЕ

**Цель:** Автоматическая генерация оглавления из заголовков

**Технические детали:**
- Панель: боковая панель слева/справа
- Синхронизация: клик по пункту = прокрутка к заголовку
- Уровни: поддержка всех уровней заголовков (H1-H6)

**Реализация:**
1. **Новый модуль:** `src/renderer/modules/toc.js`
2. **Функции:**
   - `parseHeadings(content)` - парсинг заголовков
   - `generateTOC(headings)` - генерация HTML списка
   - `scrollToHeading(id)` - прокрутка к заголовку
   - `updateTOC()` - обновление при изменениях

3. **UI компоненты:**
   - Переключаемая боковая панель
   - CSS: `src/renderer/styles/components/_toc.css`
   - Кнопка в toolbar: "Оглавление"

4. **Интеграция:**
   - Парсинг Markdown в `preview.js`
   - Синхронизация скролла
   - data-атрибуты для навигации

**Оценка времени:** 3-4 дня
